# Movement Simulator

Movement simulator based on Crab Game, Spiderman, and Apex Legends

Features:
	Wall Jumping,
	Dashing,
	Grappling Hook,
	Hitscan Shooting,
	Inventory System,
	Dialougue System,
	2000s Graphics
	
Keys: WASD (movement), Space (Jumping, Wall Jumping), E (Dashing), I (Inventory), F (Dialouge), Left Click (Shoot), Right Click - Hold (Use Grappler)
